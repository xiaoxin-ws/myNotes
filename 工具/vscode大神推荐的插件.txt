ESLint
将ESLint JavaScript集成到VS代码中。
以下设置为包括ESLint在内的所有提供程序都启用了自动修复：
"editor.codeActionsOnSave": {
    "source.fixAll": true
}
复制代码相反，此配置仅在ESLint上将其打开：
"editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
}
复制代码您还可以通过以下方式有选择地禁用ESLint：
"editor.codeActionsOnSave": {
    "source.fixAll": true,
    "source.fixAll.eslint": false
}
